# 🎓 Teka-Teki Silang - Sejarah Pendudukan Jepang di Indonesia

Website interaktif untuk bermain teka-teki silang dengan tema sejarah pendudukan Jepang di Indonesia (1942-1945).

## 🎯 Fitur Utama

- ✓ Teka-Teki Silang Interaktif dengan grid yang responsif
- ✓ Pertanyaan bertemakan sejarah pendudukan Jepang
- ✓ Sistem petunjuk dinamis
- ✓ Validasi jawaban otomatis
- ✓ Scoring system (benar/salah)
- ✓ Responsive design (desktop, tablet, mobile)
- ✓ UI/UX modern dengan gradient dan animasi

## 📝 Soal yang Disajikan

### Mendatar
1. **BELANDA** - Penjajah Indonesia sebelum Jepang
2. **PENJAJAH** - Orang yang menguasai negara lain
3. **MERDEKA** - Keadaan bebas dari penjajahan
4. **JEPANG** - Negara penguduk Indonesia 1942-1945
5. **SOEKARNO** - Presiden pertama Indonesia

### Menurun
1. **BPUPKI** - Badan persiapan kemerdekaan Indonesia
2. **AGUSTUS** - Bulan proklamasi kemerdekaan
3. **PETA** - Pasukan militer era Jepang
4. **HATTA** - Wakil presiden Indonesia pertama
5. **WWII** - Perang dunia era pendudukan Jepang

## 🛠️ Teknologi

- HTML5
- CSS3 (Grid, Flexbox, Gradient)
- JavaScript ES6+

## ⌨️ Cara Bermain

1. Klik pada kotak untuk memilih sel
2. Ketik huruf untuk menjawab
3. Gunakan tombol panah untuk navigasi
4. Tekan Tab untuk pindah ke sel berikutnya
5. Klik tombol Periksa Jawaban untuk validasi

## 🎨 Fitur Styling

- Gradient background ungu modern
- Highlight untuk sel aktif dan kata terpilih
- Color feedback (hijau=benar, merah=salah)
- Animasi smooth pada interaksi
- Fully responsive di semua ukuran layar
